ЦИЛИНДРЫ СПЕЦИАЛЬНОГО НАЗНАЧЕНИЯ

Мультипозиционные (или оппозитные) цилиндры. Рисунок C. 18. фиг.1.

Стандартный цилиндр позволяет обеспечить две позиции, определенные конечными положениями штока поршня.
Жестко соединяя два цилиндра вместе можно получить три позиции, если длины хода цилиндра одинаковые. Если длины хода различны, тогда возникают четыре различных положения. Цилиндры часто соединяются задними крышками друг к другу в одну линию используя блоки цилиндров. Однако это возможно для двух цилиндров, чтобы разделить общий шток поршня (не изображено).
Каждое положение точно определено и может быть получено повторно, присоединяя один шток поршня к детали, которая будет перемещаться. Корпуса цилиндра будут двигаться, в зависимости от того в какие полости подается давление воздуха..
Рисунки a) и b) представляют последовательные позиции для каждого случая. Стрелки указывают полости, в которые подается воздух, чтобы получить требуемое положение.

Тандемный цилиндр. Фиг. 2.
Сила, развиваемая цилиндром зависит от давления, приложенного к площади поверхности поршня. Когда требуется значительная сила, но пространство для цилиндра с большим диаметром поршня ограничено, должна использоваться альтернатива. Удачное решение - тандемный цилиндр, состоящий из отдельного блока-корпуса с двумя положительными и отрицательными полостями и двумя поршнями, которые соединяются общим штоком поршня.
Сила, которая может быть получена, определяется суммой площадей поверхности обоих поршней и действующим давлением. Можно иметь большее количество последовательных поршней, чтобы получить большие силы.

Цилиндры с двухсторонним штоком. Фиг. 3.
Цилиндр имеет два оппозитных штока, связанных с общим поршнем, каждый из которых имеет опору и уплотнение в крышках цилиндра с обеих сторон. Каждая полость цилиндра имеет равный объем и равные площади поверхности с каждой стороны поршня. Это полезно, когда требуются равные силы в обоих направлениях. Другие преимущества цилиндра этого типа:
a) В длинноходовых цилиндрах увеличенное расстояние между опорами (крышками цилиндра) уменьшает деформации штока и минимизирует износ направляющей втулки и грязесъемной манжеты.
b) Когда требуется установить датчик в средней позиции и-или в концах хода, но нет свободного пространства для них в рабочей зоне, может использоваться второй шток для установки управляющих кулачков.

Цилиндры с тросовым штоком. Фиг. 4.
В этих цилиндрах шток поршня заменен тросом, выходящим из блоков с двух сторон и соединенный с перемещаемым звеном механизма через два шкива. Преимущества этого типа конструкции - уменьшенная площадь основания привода, по сравнению с подобным нормальным цилиндром, длина которого удваивается, когда шток поршня выдвинут.
Такие цилиндры часто используются в конструкциях с большой длиной хода, в условиях ограниченного пространства, например при использовании для скользящих дверей.
