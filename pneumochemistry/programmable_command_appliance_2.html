УПРАВЛЕНИЕ ПИТАНИЕМ ВОЗДУХА. ПРОГРАММИРУЕМЫЙ КОМАНДОАППАРАТ - Часть II

Зная последовательность и назначение каждого сигнала распределителей конца хода, синтез схемы упрощается: для демонстрации этого, анализируется следующая последовательность. А+/A-/B+/С+/C-/D+/D-/B-, она уже рассматривалась в T.C.42 с применением "каскадной техники".
Выход из восьми сигналов конечного положения, пять из которых блокирующие, не вызывает сложности, потому что программируемый командоаппарат позволяет не учитывать этот факт; необходимо только установить число требуемых блоков.
При рассмотрении вышеприведенной последовательности нет никаких одновременных движений цилиндров, таким образом, получается восемь шагов и столько же блоков.
Однако, перед осуществлением соединений, необходимо знать назначение каждого сигнала, поступающего с распределителей конечного положения. Процедура простая; для определения структуры "программируемого командоаппарата", достаточно следовать за порядком движений, записывая рядом обозначение распределителя, который срабатывает в конце хода, помня, что каждое движение используется для следующего хода.
Как пример рассмотрим, что происходит в 1-ом блоке, это повторно описывается ниже:
фиг.1. T.C. 44

- Первое движение А+, поэтому выход U1 воздействует на соответствующий силовой распределитель; входной сигнал блока – a1, то есть тот, который поступает от распределителя определяющего окончание выполнения движения А+, Этот сигнал, используется для следующего движения, которое определяется блоком №2.
Полный перечень движений представлен следующим образом:
1-ый блок U1 = А+I1 = а1
2-ой блок U2 = А-I2 = а0
3-ий блок U3 = В+I3 = b1
4-ый блок U4 = С+I4 = с1
5-ый блок U5 = С-I5 = с0
6-ой блок U6 = D+I6 = d1
7-ой блок U7 = D-I7 = d0
8-ой блок U8 = B-I8 = b0
Здесь на основании правильно записанных связей можно начать строить схему, при условии, что все распределители конечного положения подключены к сети и что выход Z последнего блока связан с бистабильным 3/2 НЗ распределителем с ручным управлением, который определяет тип цикла: непрерывный или одиночный.
Кроме того, IC- распределитель, также 3/2 НЗ, но моностабильный, должен быть связан через ИЛИ с первым фитингом Z для начала цикла:
- в начале каждого нового цикла при автоматическом режиме;
- после каждого сброса схемы;
- каждый раз при одиночном цикле.
Заметим, на наличие пяти блокирующих сигналов не влияет на последовательность выполнения программы движения цилиндров. Это видно и на шаговой на пошаговой диаграмме, и в логических уравнениях, которые связывают движения с сигналами. Все это позволяет устройству задания последовательности управлять сигналами, поступающими с распределителей конечного положения, внутри этого устройства, причем независимо от продолжительности сигналов
В заключении, отметим, что устройство задания последовательности - программируемый командоаппарат игнорирует блокирующие сигналы. При использовании других методов синтеза блокирующие сигналы являются причинами целого ряда проблем пневматической системы автоматизации.
Следует также заметить, что при использовании устройства задания последовательности, снижаются последствия случайного срабатывания распределителей конечного положения, так как всегда только один из них активен. Несмотря на вышеупомянутые преимущества, широкое применение подобных систем имеет некоторые ограничения из-за высокой стоимости отдельных блоков и задержки действия сигнала, поскольку он вынужден проходить значительные расстояния.
Альтернативным вариантом управления блокирующими сигналами является импульсный генератор. Он уже был описан в примере, касающемся задержек сигналов; где каждый блокирующий сигнал должен питать элемент НЕ и схему задержки. Если требуется использовать такой метод, необходимо проверить, чтобы отключаемые блокирующие распределители конца хода (между циклами) обеспечивали достаточно быстрый выхлоп воздуха из системы задержки. В ином случае потребуется установить клапан быстрого выхлопа выше дросселя.
