КОМАНДА СТАРТ ЦИКЛА

Под "Командой старт цикла" (I.C.) предполагается действие, которое оператор производит каждый раз, для запуска цикла. Это действие может быть ограниченным по времени или длительным, в зависимости от приложения.
- При полуавтоматическом цикле, действие должно повторяться для каждого цикла. В этом случае распределитель I.C. должен быть моностабильным.
- Для автоматического цикла, действие требуется только вначале, так как последовательность перемещений и повторения циклов происходит автоматически вследствие сигналов распределителей конца хода.
I.C. распределитель в этом случае должен быть бистабильным по двум причинам:
- Устранение необходимости в операторе, для обеспечения начала каждого цикла.
- Прерывание действия и восстановления условия "старта" системы. Следовательно, новый сигнал "F.C." определяется как конец цикла.
Для обеих этих причин, I.C./F.C. распределитель должен быть подключен последовательно с последним сигналом, который выдавался распределителем конечного положения в конце каждого цикла.
Этот сигнал легко может быть определен на пошаговой диаграмме (см. рис. Т.С.24, фиг.1):

- Из четырех распределителей, которые принимают информацию о достижении конечных положений цилиндров, определяют те, которые вызывают действия для начала каждого цикла: a1 для цилиндра А и bo для цилиндра В.
- Рассмотрим ситуацию конца цикла (шаг 1, который следует после шага 4), обратим внимание на распределитель aО, который срабатывает последним и используется для продолжения цикла.
Распределитель I.C. определяет действие этого сигнала, то есть цикл может начинаться или продолжаться только, когда есть сигнал I.C., объединенный с последним выдаваемым сигналом (в этом случае аО) с использованием клапана И, и является логическим выражением условия старта.
Возможные решения связаны с расстоянием между устройством контроля положения и распределителем, который должен принять сигнал. Чем более длинная линия, тем медленнее и слабее действие сигнала.

Рассмотрим соединения:
Фиг.2 и 3 - Последовательное соединение, которое устанавливает функцию И между распределителем конечного положения и 3/2 НЗ бистабильным распределителем с ручным управлением выбора I.C/F.C;
- Они по-разному питаются от сети, один получает питание от другого. При функционировании, в соединяющем тракте, происходят следующие ситуации:
- Фиг.2: Сжатый воздух с распределителя конечного положения поступает на I.C. распределитель, только при включенном а0, то есть в конце каждого цикла;
- Фиг.3: Трубопровод постоянно находится под давлением; выход воздуха из Х происходит только через распределитель конечного положения.
Фиг.4 - С введением клапана И, два предыдущих распределителя приобретают независимое питание, каждый непосредственно соединен с сетью;
- Если управляющий распределитель находится в положении I.С. (Р соединено с А), сигнал всегда присутствует на клапане И;
- Выход X принимает состояние ЕДИНИЦА при каждом воздействии на распределитель конечного положения.
Фиг.5 - Соединение такое же как в примере на фиг.4, с разницей в том, что два моностабильных распределителя замещают один управляющий бистабильный распределитель, чтобы разделить действия I.C. и F.C.
- Это требует введения 3/2 НЗ распределителя с двухсторонним пилотным управлением для запоминания сигналов, поступающих с двух управляющих распределителей.
Это делает возможным разделить управляющие действия и в других случаях, как на фиг.2 и 3.
В заключении можно спросить: Когда оператор должен вмешаться, чтобы прервать цикл? На любом шаге, не важно, случается ли это в начале или в конце цикла.
Цикл будет закончен в любом случае, потому что другой распределитель конечного положения имеет питание независимое от управляющего прерыванием цикла распределителя.
