Для того, чтобы сделать положительное заключения по данной технике устранения блокирующих сигналов, используем предыдущую последовательность, как исходную, для описания процесса ее преобразования в схему.

Для простоты анализа ниже представлены логические уравнения каждого движения и результаты разделения на группы:


A+=IС x b0 x U3 1-я 2-я 3-я
A-=a1 A+ A-B+C+ C-D+ D-B-A+ A-
B+=a0 x U1 a1 a0 b1 c1 c0 d1 d0 b0
C+=b1 x U1 S1 S2 S3
C-=c1 U1 U2 U3
D+=c0 x U2
D-=d1
B-=d0 x U3

Удобно начинать схему сверху. Силовые распределители и цилиндры условно не показываются.


1) С учетом вышеприведенной информации представлены все необходимые распределители:
- Два 5/2 распределителя памяти, потому что требуется три сигнала выхода
- Восемь моностабильных 3/2 НЗ распределителей конечного положения
- Один моностабильный 3/2 НЗ управляющий распределитель для старта цикла.

2) Сделаны каскадные соединения между распределителями памяти, обозначены и входы и выходы

3)Записан ход цилиндра, с учетом каждого распределителя конечного положения, на который он воздействует

4) Сигнал записан, в соответствии с каждым распределителем конечного положения

5) Не забывая о том, что схема должна представлять ситуацию начала цикла, у распределителей конечного положения, которые уже сработали, обозначается символ.

6) Только неблокирующие распределители конечного положения и нижний распределитель памяти подключен к сети.
В этом пункте можно выполнить остальную часть соединений, определяемую символами, присвоенными при делении на группы (также как показано на рисунке). Здесь такая последовательность сигналов:
а1 - Уже соединен с сетью, выдает сигнал для движения A- и для первого действия S1 на пилот Y нижнего распределителя памяти.
а0 - Вход Р соединен с U1 и выход А соединен с силовым распределителем вызывая движение B+.
b1 - Вход P соединен с U1 и выход А соединен с силовым распределителем вызывая движение С+.
с1 - Уже соединен с сетью, выдает сигнал для движения С- и для второго действия S2 на пилоте Z верхнего распределителя памяти.
с0 - Вход P соединен с U2, и выход А соединен с силовым распределителем вызывая движение D+.
d1 - Уже соединен с сетью, выдает сигнал для движения D- и для третьего действия S3 на пилоте Z нижнего распределителя памяти.
d0 - Вход P соединен с U3, и выход А соединен с силовым распределителем вызывая движение В-.
b0 – Вход Р соединен с U3, и выход А соединен с силовым распределителем вызывая движение А+ посредством распределителя IC.
В этой схеме, первый распределитель конечного положения каждой группы (a1, c1, d1), выполняет двойную функцию, воздействуя на соответствующий распределитель мощности и на каскад.
В случае малой продолжительности этот сигнал может не обеспечить выполнение обеих этих функций.
Модификация, предложенная во второй схеме, заключается в том, что распределитель конца хода используется только для распределителя памяти, в то время как функция управления распределителями мощности обеспечивается выходами соответствующего каскада.
