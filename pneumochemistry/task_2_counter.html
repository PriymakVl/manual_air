ЗАДАЧА №2 - СЧЕТЧИК

Из описания задачи и графического представления последовательности видно, что цилиндр А повторяет дважды цикл для перемещения двух рядов по три банки в зону подъема.
Следовательно, необходимо считать наличие этих двух рядов посредством датчика конечного положения a1, который определяет завершение каждого прямого хода цилиндра A.
Система счета:
- запоминает первый сигнал a1;
- использует первый сигнал a0 для перехода к следующему действию счета.
Полученный в результате счета сигнал (2a1), используется для:
- предотвращения перемещения других банок, пока предыдущие не будут удалены (при наличии этого сигнала);
- разрешения начала второй последовательности.
Каждый выходной сигнал схемы счета (сигнал 2a1) должен быть отменен перед началом нового счета.
Ряд диаграмм показывает состояние воздуха в системе счета,в соответствии с четырьмя положениями, которые цилиндр А занимает на каждом шаге цикла: T.C. 47

фиг. 1 - В этот момент, схема включает два распределителя памяти и элемент И, подключенный, как изображено на рисунке и в ситуации с воздухом, когда сигнал a1 обрабатывается первый раз.
- Сигнал a1 переключает первый распределитель памяти, подключенный к сети, который в свою очередь запитывает второй распределитель памяти и подает сигнал на первый элемент И.
фиг. 2 - Сигнал а0 проходит через первый И, переключает второй распределитель памяти, который запитывает третий распределитель памяти и подает сигнал на второй элемент И.
фиг. 3 - Второй сигнал a1 может пройти через элемент И для переключения третьего распределителя памяти; выходной сигнал 2a1 используется для вышеупомянутых функций.
фиг. 4 - Обработанный сигнал 2a1 система сбрасывает сигналами b1 и с0, которые объединены элементом И, поэтому данные сигналы гарантируют, что банки подняты.
Для начала работы системы и нового счета необходимо, чтобы первым приходил сигнал от датчика конечного положения a1; с этого момента обработка сигналов начинается снова согласно вышеприведенной последовательности.
Описывается подсчет только двух сигналов, потому что такое количество требуется посчитать в этой задаче. Для подсчета большего числа, необходимо выполнять следующие правила:
- Для каждого числа больше двух, система требует дополнительно два распределителя памяти и два элемента И.
- Если необходимо получить результат подсчета в импульсной форме, можно исключить последний распределитель памяти и использовать выходной сигнал последнего И.
Однако необходимо добавить, что при подсчете большого числа, подобная система может оказаться неэкономичной; в этом случае следует рассмотреть другие варианты организации счета на иной физической основе:
- Пневматический счетчик импульсов: в основе его заложен принцип переключающейся механической собачки.
- Электронный счетчик импульсов: пневматический сигнал должен преобразовываться в электрический посредством реле давления, а электрический выходной сигнал должен преобразовываться в пневматический сигнал посредством распределителя с электромагнитным управлением.
