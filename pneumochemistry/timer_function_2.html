Функционирование таймера - Часть 2

Действия по изменению результата сигнала не всегда требует регулировки времени: иногда достаточно задержки действия. Это изменение продолжительности времени сигнала с регулированием дросселя выполняется нечасто. Чтобы лучше понять порядок этих действий, рассмотрим два примера на распределителе с прямым управлением, используя различные компоненты для временных функций.
См. рис. Т.С.23, Фиг.1: Выход X должен быть спланирован по времени:

Здесь три элемента: - дроссель, пневмоемкость и распределитель ДА - образуют таймер, он обведен тонкой линией.
Действие входного сигнала "a" проявляется на выходе через постоянное время "t" по следующим двум причинам:
- Неизменность воздушного потока, поступающего через дроссель;
- Неизменность сопротивления между пилотом и питающей сетью.
Поэтому, переключение распределителя ДА всегда происходит "определенным" образом и в относительно "точный" момент времени, который гарантируется повторением постоянной временной задержки включения. Следовательно, сигнал на пилоте Z всегда будет переключать распределитель с прямым управление при одних и тех же условиях, независимо от реакции механизма возврата. Сигнал Х управляется непосредственно, поэтому соответственно имеется временной эффект на выходе распределителя ДА.
Фиг.2: Выход X требует задержки:
Функция, выполняемая распределителем ДА в предыдущем случае, не требуется. Устройство задержки, отмеченное тонкой линией, может быть составлено из дросселя и пневмоемкости.
При такой схеме сигнал "a" действует прямо на пилот Z распределителя. Время сигнала регулируется дросселем. Тип механизма возврата может быть различным. Однако, это не важно, поскольку в подобных задачах сигнал X не требует точного определения продолжительности времени.
Другими словами, продолжительность задержки может также управляться механизмом возврата. Например, при одинаковом расходе и объеме пневмоемкости, распределитель с пневматическим возвратом переключится с меньшей задержкой, потому что ему необходимо преодолеть только трение внутренних элементов.

Фиг.3: Формирователь импульса
- Используется, чтобы преобразовать длительный сигнал в импульс управляемой продолжительности;
- Схема проста. Сигнал "a" соединен с портом питания распределителя НЕ и с пилотом Z, через дроссель и пневмоемкость;
- Продолжительность сигнала X зависит от задержки сигнала "a" поступающего на Z;
- Диаграмма дает только закономерность связи между временами из двух сигналов.
Время "t" очевидно зависит от изменений продолжительности сигнала "a".
Фиг.4: Расширитель импульса
Обычно, эта функция реализуется с использованием бистабильного распределителя памяти, который требует двух сигналов для переключения. Когда имеется только один сигнал, то возможное решение - схема задержки.

Схема работает следующим образом:
- Сигнал "a" должен иметь "более продолжительное" действие на выходе X. Поэтому, эта ситуация противоположна примеру на фиг.3;
- Это служит причиной для появления клапана ИЛИ, который при каждом появлении сигнала "a", позволяет удерживать этот сигнал либо по "основной", либо по "дополнительной" линии:
- по основной линии сигнал должен иметь продолжительность достаточную для переключения распределителя ДА, который соединяет с сетью часть схемы, составленной из:
- Задержки
- распределителя НЕ, который дает толчок для перехода в состояние ЕДИНИЦА сигнала X, сигнал "a" по-прежнему присутствует на клапане ИЛИ.
- по дополнительному пути и для продолжительности в зависимости от установки регулятора задержки, посредством соединения X с R. Непрерывный сигнал заменяет сигнал "a", как только он пропадает.
В этом плане замыкание цепи выполняется на длительность, определяемую действием задержки на время переключения распределителя НЕ и последовательное восстановление НУЛЕВОГО СОСТОЯНИЯ X и переустановки распределителя ДА.
С этого момента из всей схемы стравливается воздух через распределитель ДА, который готовит себя к приходу следующего сигнала "a".
